# cn-portal-pages

本仓库用于归档与发布多个独立的 HTML 页面。所有页面均为静态资源，不依赖特定域名或后端服务。

## 项目简介

cn-portal-pages 是一个通用型 HTML 页面集合仓库。页面内容覆盖信息展示、工具界面、导航入口等多种形态，适合作为前端页面存档或快速部署的静态资源库。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 主要页面归档目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 存放所有独立 HTML 页面，每个页面可自包含样式与脚本。
- `assets/` 存放被多个页面引用的公共资源文件。
- 页面间无强依赖关系，可单独访问。

## 页面归档说明

- 所有页面均为纯静态 HTML，兼容主流现代浏览器。
- 页面内容不涉及具体域名、商业推广或诱导点击行为。
- 部分页面可能包含示例数据或占位内容，仅供展示布局与功能。
- 页面文件名建议使用有意义的英文或拼音命名，避免特殊字符。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器打开 `pages/` 目录下的任意 `.html` 文件即可查看。
3. 如需部署到服务器，将整个仓库内容上传至 Web 服务器根目录或子目录即可。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 以帮助改进页面质量。
- 新增页面请放入 `pages/` 目录，并在本文件目录说明中更新记录。
- 请保持页面风格简洁，避免引入过多外部依赖。
- 本仓库不承载具体网站业务逻辑，仅用于静态页面归档与展示。
